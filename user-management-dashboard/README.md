# User Management Dashboard

A React-based User Management Dashboard built using JSONPlaceholder API.

## Features

* Fetch users from JSONPlaceholder API
* Search users
* Advanced filters
* Column sorting
* Pagination
* Add user
* Edit user
* Delete user
* Form validation
* Error handling
* Loading state
* Responsive design

## Tech Stack

* React
* Vite
* Axios
* CSS

## Project Structure

```
src
├── api
├── components
├── hooks
├── utils
├── styles
```

## Installation

```bash
git clone <repository-url>

cd user-management-dashboard

npm install

npm run dev
```

## API Used

```
https://jsonplaceholder.typicode.com/users
```

## Assumptions

* Since JSONPlaceholder is a mock API, create/update/delete operations are simulated locally.
* Departments are generated programmatically.
